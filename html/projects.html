2D Game Engine

As part of a semester-long project, I am implementing a comprehensive 2D game engine from scratch using Java. My engine includes Systems to render game objects, handle user input, detect in-game collisions, and many more capabilities.

One of the very first steps I took was understanding the something of execution in order to design the frameworks upon which my games would be built.

I have built three games thus far on top of my engine:

Tic Tac Toe: my rendition of tic-tac-toe, which allows for multiple grid sizes!

Alchemy: my rendition of this delightful game

Paws off my Crops: a work in progress involving tending to, and protecting, produce
